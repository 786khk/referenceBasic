# 문자열

- 문자열은 기본 자료형이 아닌 String 클래스로 `참조형`이다.

### 리터럴

```java

String s1 = "Strng과 char는 다르다.";
String s2 = "안녕, \tJAVA의 세계에 온것을 \t환영해";

```

- 49 + 49 = 98 로 출력된다.
